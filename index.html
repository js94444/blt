🏠 미소AI

MISO에서 제공하는 채팅 형식의 GenAI 서비스입니다

🛡️

안전한 환경에서 구동

미소AI는 안전한 환경에서 구동되는 채팅 형식의 GenAI 서비스입니다. 따라서 외부 GenAI 서비스를 사용시 불편하였던 대외비 관련 걱정 없이 안심하고 사용하실 수 있습니다.

📚

사내 지식 기반 답변

미소AI는 담당자가 미리 세팅한 다양한 사내 지식들(전결규정, 복리후생규정, 업무규정집 등)을 기반으로 답변을 합니다.

💡 기존 업무 vs 미소AI 활용

❌ 기존 방식

  1. 여러 PDF 문서들을 찾기
  2. 문서를 열고 검색하기
  3. 원하는 정보가 없으면 다른 문서 찾기
  4. 반복적인 검색 과정
→

✅ 미소AI 활용

  1. 나의 상황을 설명
  2. 미소AI에게 답변 요청
  3. 가장 적절한 문서의 내용을 찾아줌
  4. 즉시 정확한 답변 획득

🔍 사내 지식 검색

미소 AI 담당자가 미리 등록해둔 사내 지식을 참조하여서 미소 AI가 답변합니다.

💼 활용 예시: 전결 규정 문의

1️⃣
상황 설명: 사내의 전결 규정이 미소AI에 등록된 경우
2️⃣
질문 방법: 나의 상황을 채팅창에 설명한 뒤 어떤 절차로 결재를 올려야 하는지 질문
3️⃣
AI 답변: 미소AI가 사내 전결 규정에 근거하여 정확한 답변 제공

📊

MISO AI 비용 구조

질문(입력 토큰)과 답변(출력 토큰)의 길이에 따라 토큰 단위로 과금됩니다. 효율적인 질문으로 월별 운영비를 최대 60% 절감 가능합니다.

💡 비용 절감 시뮬레이션

❌ 비효율적 사용

질문 1: "보고서 써줘"

질문 2: "형식도 알려줘"

질문 3: "더 자세히 써줘"

월 예상 비용: ₩850,000 (3회 반복 × 높은 토큰 사용량)

✅ 효율적 사용

한 번의 질문: "마케팅 성과 분석을 위한 10페이지 분량의 월간 보고서를 표준 비즈니스 형식으로 작성해 주세요. 포함 내용: 1) 매출 현황 2) 고객 분석 3) 향후 전략"

월 예상 비용: ₩340,000 (1회 완성형 질문 × 최적화된 토큰 사용)
💰 절약 효과: ₩510,000/월 (60% 절감)

🔸 입력 토큰

사용자가 입력하는 텍스트의 길이에 따라 계산

🔸 출력 토큰

AI가 생성하는 응답의 길이에 따라 계산

🔸 특수 기능

이미지 분석, 파일 업로드 등 추가 기능 사용 시

💡 비용 절약 팁

1

질문을 하나로 통합
여러 번 나누어 묻지 말고 한 번에 완성된 질문을 하세요

2

불필요한 설명 제외
핵심적인 내용만 간결하게 작성하세요

3

목적에 맞는 활용
업무와 직접 관련된 내용에만 사용하세요

❓ 질문 품질 제고

AI로부터 최고의 답변을 얻는 방법을 알아보세요

✅ 이렇게 하세요

좋은 예시

"마케팅 보고서를 작성해야 합니다. 2024년 3분기 매출 데이터를 바탕으로 다음 분기 전략을 포함한 10페이지 분량의 보고서 구조와 주요 내용을 제안해 주세요. 타겟 고객층은 30-40대이고, 주력 제품은 생활용품입니다."

  • 🎯 구체적인 목적 명시
  • 📊 필요한 정보 모두 포함
  • 📏 원하는 결과물 명확히 설명
  • 🔍 배경 정보 충분히 제공

❌ 피해야 할 것

나쁜 예시

"보고서 써줘"
"이거 어떻게 해?"
"좋은 아이디어 있어?"

  • ❌ 모호하고 애매한 표현
  • ❌ 정보 부족으로 인한 추가 질문 필요
  • ❌ 여러 번 나누어 질문하기
  • ❌ 목적 없는 일반적인 질문

📝 질문 템플릿

[배경 정보] + [구체적 요청] + [원하는 결과물] + [추가 조건]

예시:
[배경] 저희 회사는 중소기업용 소프트웨어를 개발합니다.
[요청] 고객 만족도 조사 설문지를 만들어 주세요.
[결과물] 10-15개 문항으로 구성된 온라인 설문 형태로
[조건] 응답 시간은 5분 이내, 객관식과 주관식 혼합

🙋‍♀️ 자주 묻는 질문

MISO AI 사용 중 자주 발생하는 궁금증들

Q

질문을 여러 번 나누어서 하면 안 되나요?

+

비용이 중복으로 발생하므로 권장하지 않습니다. 가능한 한 모든 요구사항을 하나의 질문에 포함시켜 주세요. 예를 들어 "보고서를 써주세요" 후에 "형식도 알려주세요"보다는 "A4 3페이지 분량의 마케팅 보고서를 표준 비즈니스 형식으로 작성해 주세요"가 더 효율적입니다.

Q

개인적인 업무도 AI로 처리해도 되나요?

+

회사 계정으로 제공되는 AI는 회사 업무에만 사용해야 합니다. 개인적인 문서 작성, 학습, 취미 관련 질문 등은 피해주세요. 업무 효율성 향상과 직접적으로 관련된 활용에 집중해 주시기 바랍니다.

Q

AI가 제공한 정보를 그대로 사용해도 되나요?

+

AI의 답변은 참고용으로 활용하고, 반드시 내용을 검토하고 사실 확인을 해주세요. 특히 법적 사항, 재무 정보, 기술적 세부사항 등은 전문가의 검토를 받으시기 바랍니다. AI는 보조 도구일 뿐, 최종 의사결정은 항상 인간이 해야 합니다.

Q

어떤 종류의 파일을 업로드할 수 있나요?

+

일반적으로 텍스트, 이미지, PDF, Excel 파일 등을 업로드할 수 있습니다. 단, 기밀 정보나 개인정보가 포함된 파일은 업로드하지 마세요. 업로드 전에 파일 내용을 점검하여 민감한 정보가 포함되어 있지 않은지 확인해 주세요.

Q

비용을 아끼려면 어떻게 해야 하나요?

+

1. 질문을 미리 정리하여 한 번에 완성된 요청을 하세요
2. 불필요한 설명이나 반복을 피하고 핵심만 간결하게 작성하세요
3. 명확하고 구체적인 요청으로 추가 질문을 줄이세요
4. 업무와 직접 관련된 내용에만 집중하세요

Q

사내 지식 검색은 어떻게 활용하나요?

+

미소AI는 사내 전결규정, 복리후생규정, 업무규정집 등이 미리 등록되어 있습니다. 예를 들어 "출장비 신청 절차를 알려주세요" 또는 "연차 사용 규정이 궁금합니다"와 같이 구체적인 상황을 설명하면, 관련 규정을 찾아서 정확한 답변을 제공합니다.
